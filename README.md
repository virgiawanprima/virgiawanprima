# Hi, I'm Virgiawan Prima Rizky

**Data Engineering · Machine Learning Engineering**

I build end-to-end data systems — from raw ingestion and warehouse modeling
to model training, deployment, and monitoring in production.

On the data engineering side, my work covers designing reliable ETL/ELT
pipelines, modeling data warehouses and lakehouses, orchestrating batch and
streaming workflows, and enforcing data quality at every stage of the pipeline.

On the ML engineering side, I focus on what happens after a model is trained:
packaging, serving, monitoring for drift, and building the pipelines that make
retraining reproducible and auditable. I treat model deployment the same way I
treat data pipelines — it needs to be observable, testable, and maintainable.

The overlap between these two disciplines is where I work best. Clean,
well-modeled data is a prerequisite for reliable ML systems, and reliable ML
systems depend on infrastructure that is built with the same engineering
rigor as the rest of the stack. That end-to-end ownership — from the first
byte of raw data to a model serving predictions in production — is what I
focus on building.
